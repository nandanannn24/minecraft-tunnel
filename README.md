# Minecraft Java 1.21.10 Fabric Tunnel Server

Tunnel server untuk multiplayer Minecraft Java dengan mod Fabric.

## Cara Setup

1. Deploy ke Railway
2. Jalankan client.py di komputer dengan Minecraft server
3. Teman connect ke URL Railway

## Environment Variables

- `MINECRAFT_HOST`: IP Minecraft server (default: localhost)
- `MINECRAFT_PORT`: Port Minecraft (default: 25565)
